# MoveOn

MoveOn — Android-приложение для поиска людей для совместных спортивных активностей.

Приложение позволяет создавать спортивные события, находить единомышленников, общаться с участниками и строить маршруты для тренировок. Помимо списка мероприятий, пользователю доступна интерактивная карта со спортивными площадками и событиями, а также возможность сохранять построенные маршруты и просматривать маршруты других пользователей.

## Быстрый запуск

### Требования

* Docker
* Docker Compose

### Запуск сервера

В корне проекта выполните:

```bash
docker compose up --build
```

После запуска будут автоматически подняты:

* Backend (Ktor)
* PostgreSQL

Сервер станет доступен по адресу:

```text
http://localhost:8080
```

### Запуск Android-приложения

1. Откройте проект в папке AndroidApp/MoveOn в Android Studio.
2. Дождитесь завершения синхронизации Gradle.
3. Если нужно, укажите адрес запущенного backend-сервера (В приложении уже указан адрес задеплоенного сервера).
4. Запустите приложение на эмуляторе или физическом устройстве.

## Возможности

* регистрация и авторизация пользователей;
* создание спортивных событий;
* поиск мероприятий по названию и фильтрам;
* присоединение к событиям других пользователей;
* просмотр событий на карте;
* создание событий с маршрутами;
* просмотр маршрутов, созданных другими пользователями;
* чаты событий;
* рейтинги пользователей после совместных тренировок;
* push-уведомления о событиях и отзывах.
